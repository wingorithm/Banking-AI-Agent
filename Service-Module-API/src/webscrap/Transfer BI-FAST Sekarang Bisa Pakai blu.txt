Transfer BI-FAST Sekarang Bisa Pakai blu
27 October 2023
Buat kamu sobatblu yang sering transfer antar bank, ada kabar baik nih buat kamu karena sekarang blu bisa kamu gunakan untuk transfer via BI-FAST. Dengan BI-FAST, nasabah hanya dikenakan biaya transfer sebesar Rp2.500 untuk transaksi transfer domestik ke bank lain.
Apa Itu BI-FAST?
Nah sebelum kamu menggunakan BI-FAST, ada baiknya kamu mengetahui apa sih BI-FAST itu?
BI-FAST adalah infrastruktur Sistem Pembayaran ritel nasional yang dapat memfasilitasi pembayaran ritel secara real-time, aman, efisien, dan tersedia setiap saat (24/7).
Layanan BI-FAST sudah bisa kamu akses di aplikasi blu melalui menu transfer dengan minimum transaksi sebesar Rp5.000 dan maksimal Rp250.000.000
Bagaimana Cara Transfer via BI-FAST di blu?
Untuk menggunakan BI-FAST di blu, yuk ikuti tahap ini!
Pilih menu Transaksi
Pilih menu transfer
Pilih Transfer Baru
a. Kalau kamu pilih 'Transfer Via Rekening', ikuti langkah ini:
Input no. rekening tujuan/penerima
Input nominal dan pilih Tipe Transfer BI-FAST
Pilih Tujuan Transaksi dan input Catatan (jika ada), lalu pilih 'Lanjut'
Konfirmasi, klik 'Lanjut' jika sudah selesai
Input PIN transaksi blu
Transaksi selesai
b. Kalau kamu pilih 'Transfer Via Proxy', ikuti langkah ini:
Input no. HP atau email tujuan/penerima yang sudah didaftarkan sebagai Proxy Address, lalu pilih 'Cek'
Input nominal, lalu pilih Tujuan Transaksi dan input Catatan (jika ada), kemudian pilih 'Lanjut'
Konfirmasi, klik 'Lanjut' jika sudah sesuai
Input PIN transaksi blu
Transaksi selesai
Mudah kan sobatblu? BI-FAST ini cocok banget buta kamu yang sering transfer antar bank karena biaya adminnya lebih hemat. Jika kamu memiliki pertanyaan mengenai Proxy Address, atau mengalami kendala, maka kamu bisa liat info lebih lengkapnya di sini atau hubungi 1500668 atau Twitter @haloblu.
Tunggu apalagi? Yuk buka bluAccount kamu sekarang buat transfer via BI-FAST dan transaksi harian lainnya!
